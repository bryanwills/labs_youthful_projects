soundfonts

Soundfonts are a collection of recorded instrument sounds that can be used to play a midi file. Many sites that used to host soundfonts aren't available anymore. So, on this page I host some of the better soundfonts I was able to find.

These are compressed in the rar format. In Windows, I recommend the program WinRar. In Linux, the package unrar is easy to get if you don't have it already. After running one of sudo apt-get install unrar or sudo yum install unrar, your archive manager should be able to uncompress rar files. Each rar file contains the soundfont as well as a text file with the source or author information.

To use, just uncompress the rar file, and in bmidi to wave, click on Choose Soundfont from the Audio menu, and point it to the .sf2 file on your hard drive.
