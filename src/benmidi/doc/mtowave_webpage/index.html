about

bmidi to wave can be used to:

  • Turn midi files into great-sounding wav and mp3 files.
  • Play midi files, especially if you are unsatisfied with your current midi out quality.
  • Get information about SoundFont files and preview their voices.

Find high quality SoundFonts online, and play each instrument in the midi with the SoundFont of your choice, resulting in the best possible sound. Use the mixer to fine-tune the volume and pan of each channel. Also, you can see a score view of notes in a particular track.

bmidi to wave is a frontend for the program timidity++, which does the actual playback. This program provides a graphical interface and saves you from having to edit configuration files.

bmidi to wave is written by Ben Fisher and is released under the GPLv3 license.

features

Render midi to wav:

  • Saves midi files to standard .wav audio.
  • Supports 16bit or 24bit stereo audio at 44100Hz.
  • Low pass filter and reverb options.

Adjust until it sounds right:

  • Start playback from anywhere in the midi
  • Mix and match soundfonts, even choosing which voices within a soundfont to use.
  • Change midi instruments
  • Change volume, pan, transposition for each channel
  • Change tempo
  • Convert format 0 to format 1
  • Save any of these modifications back to a midi file

Get information:

  • See musical score view of a track
  • See track and channel information
  • Open list of midi events.
  • SoundFont information tool shows presets within a soundfont and can preview them.
